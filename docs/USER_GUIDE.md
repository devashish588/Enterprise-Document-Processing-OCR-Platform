# User Guide

## Sign In

Open the frontend and use an email/password. The local demo registers the account if it does not exist.

## Upload

Go to Upload and select a document. Text files are useful for smoke testing. Images use Tesseract/OpenCV when the local binaries and packages are installed.

## Review

Open a document from the dashboard to view OCR text and extracted fields.

## Search

Use the Search page to find documents by filename or OCR text.

## Export

Use Reports to download a CSV summary of processed documents.
